- Reports not refreshed daily, causing delays in decision-making
- High operational costs due to multiple tools and environments\n\nSolution: - Implemented a fully connected data and reporting platform using Microsoft Fabric
- Integrated data from four primary systems via Data Pipelines and Notebooks
- Established a dedicated analytics warehouse within Microsoft Fabric
- Automated daily and weekly data refreshes using Fabric’s scheduling features
- Created a unified semantic model for consistent KPIs and metrics
- Migrated reports to Power BI within Microsoft Fabric for consolidated reporting
- Applied role-based access control aligned with compliance requirements\n\nSolution Benefits: - Delivered over 70 government-mandated Jawda reports within one month
